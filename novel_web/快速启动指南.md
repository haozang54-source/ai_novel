## ⚠️ 如果遇到问题

### 问题1: 404 错误
- 确保后端服务已重启
- 检查后端日志是否有错误

### 问题2: 500 错误
- 检查数据库是否创建成功
- 重新运行迁移脚本：
  ```bash
  cd /Users/zelaszang/Documents/UGit/ai_novel
  pipenv run python novel_web/backend/migrate.py
  ```

### 问题3: CORS 错误
- 确保前端运行在 http://localhost:5173
- 确保后端运行在 http://localhost:5001
